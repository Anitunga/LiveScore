This project, titled "BasketBall LiveScore," involves creating a .NET and Web-based application to encode and track basketball game details in real-time. 

Key functionalities include:
*Game Setup: Input teams, players, player numbers, quarters, quarter durations, timeout durations, and starting players.
*Real-Time Game Events: Track scores (1, 2, or 3 points), player fouls, substitutions, game clock status (start/stop), quarter changes, and timeouts.

The goal is to provide an intuitive interface for recording and managing game data, allowing for accurate, live game tracking.
